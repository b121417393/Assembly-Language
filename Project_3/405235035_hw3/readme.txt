以下介紹各個暫存器的用途及存放的內容

用來存放位置：
r0用來存放矩陣A的起始位置
r3用來存放矩陣B的起始位置
r9用來存放矩陣C的起始位置

用來存放矩陣的值：
r1用來存放矩陣A第n列第1行的值 (n,1)
r2用來存放矩陣A第n列第2行的值 (n,2)
r4用來存放矩陣B第1列第m行的值 (1,m)
r5用來存放矩陣B第2列第m行的值 (2,m)

用來存放計算出的新值：
r6用來存放(n,1)乘(1,m)的值
r7用來存放(n,2)乘(2,m)的值
r8用來存放r6加r7的值，此值為矩陣C(n,m)的新值，鎖底會在計算後存放回矩陣C的位置

這個程式先計算出矩陣C第一列的值，再使用branch的指令，繼續計算第二列、第三列的值，之後結束迴圈

在程式的最後 r1暫存器會指向矩陣C的起始位置

使用程式前請先將arm-none-eabi-gcc的所在位置加入環境變數的絕對路徑PATH之中 (否則makefile需要修改成絕對路徑的寫法)
之後執行make即可產生hw3.exe
